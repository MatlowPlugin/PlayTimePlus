# PlayTime+

PlayTime+ is a Minecraft plugin that rewards players based on their active playtime.

## Features
- Real-time playtime tracking
- Configurable reward milestones
- Interactive GUI with live updates
- Pending rewards system
- Advanced anti-AFK detection
- No database required

## Commands
/playtime  
/playtime reset <player>  
/playtime reload  

## Permissions
playtimeplus.use  
playtimeplus.admin
